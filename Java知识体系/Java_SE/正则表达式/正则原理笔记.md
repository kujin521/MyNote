# 正则表达式

> 通过指定字符串模式 指定特定字符

## 保留字

| 字符  | 说明                         |
| ----- | ---------------------------- |
| .     | 匹配任意单个字符             |
| *     | 前面的表达式可重复0次或多次  |
| +     | 重复1次或多次                |
| ?     | 表示0次或1次                 |
| {n,m} | 指定范围次数                 |
| ()    | 分组 组是根据左括号 依据下标 |
| ^ $   | 表达式开始和结束标记         |
| \|    | 或                           |
|       |                              |

## 找到一个匹配

```java
String regx="[+-]?\\d+";
Char Sequence intput=...;
```

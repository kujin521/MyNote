## chmod命令

chmod [选项] 模式 文件名

选项

​	-R 递归

模式

​	[ugoa] [+-=] [rwx]

​	[mode=421]

### 例子（权限编辑） 

- chmod u+x a.txt

  所有者可以执行

- chmod g+w,o+w a.txt

  所属组可以写 其他组可以写

- chomd a=rwx a.txt

  所有人可以读写执行

### 例子 （利用数字）

- chmod 755 a.txt

  所有者7读写执行

  所属组5读执行

  其他人5读执行

## 常见数字权限

后面的数值不能大于前面的数值（其他人的权限不能大于自己的权限）

| 权限 | 描述                               |
| ---- | ---------------------------------- |
| 777  | 最高权限                           |
| 644  | 文件权限（读写 读 读）             |
| 755  | 执行权限（读写执行 读执行 读执行） |

## Nginx简介
	Nginx是什么？能干什么？有什么特点？
1. 轻量级web服务器，也是一款反向代理服务器
2. 可直接支持Rails和PHP的程序
3. 可作为http反向代理服务器
4. 负债均衡服务器
5. 邮件代理服务器
6. 帮助前端动静分离
## Nginx命令
	1. 启动systemctl start nginx
	2. 停止systemctl stop nginx
	3. 状态systemctl status nginx
	4. 重启systemctl reload nginx

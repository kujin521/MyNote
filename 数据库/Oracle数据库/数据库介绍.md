# 数据库(Database DB)

> 长期存储在计算机内有的有结构 大量的 共享的数据集合

- 长期: 持久存储
- 有结构 存储数据量大
- 共享 多应用通过数据库共享

# 数据库类型

- 关系数据库
  - MySQL
    - MariaDB(MySQL替代品)
  - PostgerSQL
  - SQL Server
  - Oracle数据库

- 非关系数据库NoSQL
  - 面向检索的列式存储Column-Oriented
    - BigTable
    - HaBase
  - 面向高并发的缓存存储Key-value
    - Redis
  - 面向海量数据访问的文档存储
    - MongoDB
    - CouchDB

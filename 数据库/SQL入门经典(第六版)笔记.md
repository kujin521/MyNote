# SQL定义

SQL表示结构化查询语言

sql命令主要类别

- DDL 数据定义语言
  - create table
  - alter table
  - drop table
  - create index
  - alter index
  - drop index
  - create view
  - drop view
- DML 数据操作语言
  - insert
  - update
  - delect
- DQL 数据查询语言
  - select
- DCL 数据控制语言
  - alter password
  - grant
  - revoke
  - create synonym
- DAC 数据管理语言
  - start audit
  - stop audit
- TCC 事务控制命令

4个事务控制命令

- commit 提交事务
- rollback 撤销事务
- savepoint 创建事务点
- set transactions 设置事务名称



使用客户端连接会登录服务器 直接连接到数据库

使用web时会登录达到数据库的互联网上



如果某个字段被定义为null 表示该字段可以不必输入任何内容

如果字段被定义not null 则表示该字段必须输入数据

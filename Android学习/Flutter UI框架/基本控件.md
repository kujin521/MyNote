## 容器 Container 相当于div

```
Container(
  child: const Text(
    'Container 背景渐变色 边框设置',
    style: TextStyle(fontSize: 40.0),
  ),
  alignment: Alignment.topLeft,//对齐方向
  //width: 500.0,
  //height: 400.0,
  //color: Colors.blue,//Color不能和decoration一起使用
  padding: const EdgeInsets.all(10.0),
  //margin: const EdgeInsets.all(10.0),
  decoration: const BoxDecoration(
    gradient: LinearGradient(transform: GradientRotation(10.9), //渐变方向
        colors: [
          //渐变颜色
          Colors.lightBlue,
          Colors.greenAccent,
          Colors.purple
        ]),
    border: Border(
      //边框颜色
      top: BorderSide(width: 5.0, color: Color(0xFFF31616)),
      left: BorderSide(width: 4.0, color: Color(0xFFDFDFDF)),
      right: BorderSide(width: 3.0, color: Color(0xFF1D1DDC)),
      bottom: BorderSide(width: 2.0, color: Color(0xFF29DE32)),
    ),
  ),
)
```

## 文本框

```dart
Text(
            '文本框,学习Flutter!!本框,学习Flutter!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!',
            textAlign: TextAlign.center, //文本对齐
            maxLines: 1, //最大行
            overflow: TextOverflow.ellipsis, //文本溢出
            style: TextStyle(
              fontSize: 25.0, //文本大小
              color: Color.fromARGB(255, 255, 125, 124), //文本颜色
              decoration: TextDecoration.underline, //下划线
            ),
          )
```

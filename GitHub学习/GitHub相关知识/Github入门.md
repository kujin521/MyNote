## 怎么用Github?

### 设置git

下载安装git

1. 打开 Git Bash。

2. 设置 Git 用户名和邮箱：

   ```shell
   $ git config --global user.name "Mona Lisa"
   $ git config --global user.email "email@example.com"
   ```

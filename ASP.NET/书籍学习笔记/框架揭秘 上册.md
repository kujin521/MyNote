> 下载对应的.NET Core SDK IDE 版本
>
> dotnet.exe 是主要的命令工具

**dotnet**主要命令

```cmd
# 查看信息
dotnet --info
# 获取脚手架模板列表(可自定义模板)
dotnet new --list
# 创建一个控制台程序
dotnet new console -n helloword
# 编译项目
dotnet build
# 运行项目
dotnet run
```

## 容器的基本用法

bean是spring最核心的东西 如果spring是水桶 那么bean就是容器里的水

定义一个**bean**

```
pulic class MyTestBean{
	private String testStr="testStr";
	publiv void setTestStr(String testStr){
		this.testStr=testStr;
	}
	public String getTestStr(){
		return this.testStr;
	}
}
```

spring 配置Bean



## 功能分析

1. 读取配置文件
2. 根据配置文件找到对应的类，并实例化
3. 调用实例化后的实例
